# Revenue Analysis Dashboard

## Overview

This project is a Revenue Analysis Dashboard built using Power BI to analyze sales performance and profitability. The dashboard converts raw sales data into interactive visualizations that help identify revenue trends, customer behavior, and product performance.

## Tools Used

* Power BI
* Power Query
* DAX
* Excel Dataset

## Key Insights

* Revenue analysis by product category
* Customer age group contribution to revenue
* Regional sales performance
* Profit and profit margin analysis

## Dataset Features

The dataset includes fields such as customer age, gender, country, product category, quantity, unit cost, unit price, revenue, and profit.

## Purpose of the Project

The goal of this project is to practice data analysis and visualization by transforming raw data into meaningful business insights using Power BI.
